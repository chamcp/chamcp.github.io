# 世界杯预测追踪

这是一个可以部署到 GitHub Pages 的静态网页。

## 文件说明

- `index.html`: 页面和交互。
- `data.js`: 赛程、每日预测、复盘数据。以后更新数据主要改这个文件。
- `CNAME`: 自定义域名，当前设置为 `chamcp.com`。

## GitHub Pages 建议

用 GitHub 用户名 `chamcp` 创建仓库：

```text
chamcp.github.io
```

把本目录里的 `index.html`、`data.js`、`CNAME` 上传到仓库根目录。开启 GitHub Pages 后，先访问：

```text
https://chamcp.github.io
```

域名解析完成后，再访问：

```text
https://chamcp.com
```
